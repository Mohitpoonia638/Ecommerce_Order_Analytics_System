# Ecommerce Order Analytics System

## Project Overview

The Ecommerce Order Analytics System is an end-to-end data analytics project built using Python, Pandas, SQLite, and SQL.

The project simulates a real-world e-commerce environment by generating synthetic datasets, introducing data quality issues, cleaning the data, loading it into SQLite, performing SQL analysis, generating business reports, and validating the system with edge-case testing.

---

## Features

- Synthetic Data Generation
- Data Quality Issue Injection
- Data Cleaning using Pandas
- SQLite Database Integration
- Basic SQL Analysis
- Intermediate SQL Analysis
- Advanced SQL (CTEs & Window Functions)
- Business Report Generation
- Edge Case Testing
- Menu Driven Execution
